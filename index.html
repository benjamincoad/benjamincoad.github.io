A Python CLI tool that efficiently detects and manages duplicate files. Features include smart size-based pre-filtering, SHA-256 hash verification, and customizable directory ignore patterns. Helps you reclaim disk space while ensuring data safety.

dtree - Directory Tree Visualizer

A Python CLI tool that generates a visual tree representation of directory structures. Features customizable ignore patterns for excluding specific directories, pretty-printed formatting with Unicode characters, and intelligent sorting with directories first.
